# 🏆 World Cup 2026 Tracker

A web app to predict and track the results of the 2026 FIFA World Cup.

## Features

- **Groups** — Browse all 12 groups and their teams, with live points
- **Matches** — Enter scores for completed group stage matches
- **Predictions** — Make your own predictions for upcoming matches
- **Standings** — View full group standings (P/W/D/L/GF/GA/GD/Pts) with qualification indicators

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no build tools
- Hosted via GitHub Pages
- Responsive, dark/light mode

## Setup

1. Clone the repo
2. Open `index.html` in your browser — or enable GitHub Pages
